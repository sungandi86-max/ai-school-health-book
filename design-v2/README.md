# design-v2 — 편집 디자인 시안 (대표 페이지 6장)

`book-source/book-source-final.md`만을 원고(SSOT)로 사용해 처음부터 새로 편집 디자인한 시안입니다. 기존 `pdf/book-final-pass2.pdf`와 그 조판 코드는 참고하지 않았습니다.

## 파일

- `book-v2-samples.pdf` — 대표 페이지 6장 (A5)
- `samples/` — 각 페이지 PNG 미리보기
- `design_v2.py` — 시안 생성 스크립트 (레퍼런스 스타일 검토 후 새로 작성)
- `fonts/` — 스크립트 실행에 필요한 서체 (Noto Sans KR 서브셋)

## 대표 페이지 6장

1. `01-book-howto.png` — 책 사용법
2. `02-chapter-opening.png` — Chapter 오프닝 (Chapter 01)
3. `03-story.png` — 이야기 페이지 (Chapter 01 도입부)
4. `04-project-screen.png` — 실제 프로젝트 화면 페이지 (IMG-001)
5. `05-workflow.png` — Workflow 페이지 (IMG-002, 책 전체 Workflow)
6. `06-practice.png` — 실습 페이지 (실습 17)

## 디자인 원칙

- 화이트 기반 + 딥네이비 포인트 + 연블루 포인트
- 이야기 · 설명 · 프로젝트 화면 · 실습 네 가지 페이지를 레이아웃으로 완전히 구분
- 소프트 카드(연블루, 테두리 없음, 둥근 모서리)를 정보 단위로 사용
- 이미지는 페이지 폭의 80~90% 이상 사용, 내부 표기(IMG-XXX) 노출 없음
- 여백은 빈 공간이 아니라 호흡으로 사용

승인 전에는 이 6장 외에 전체 255페이지를 재조판하지 않습니다.
